# DentalClinicApp

Objetivo: <br> 
Crear un software para agendar consultas, revisiones y exámenes en una clínica odontológica, con dos doctores trabajando con los mismos horarios.<br> <br> 

Tecnologías:<br> 
✅Backend: .NET (C#)<br> 
✅Frontend: WPF<br> 
✅Base de datos: PostgreSQL<br> 

Funcionalidades:<br> 
✅Agendar citas por fecha y hora <br> 
✅Ver la disponibilidad de cada doctor <br> 
✅Tener un calendario para la planificación <br> 
✅CRUD (Crear, Leer, Actualizar, Eliminar) para las citas <br> 
✅Ingreso de datos como nombre del paciente, tipo de consulta, doctor asignado v
