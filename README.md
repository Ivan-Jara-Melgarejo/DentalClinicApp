# DentalClinicApp

Objetivo: 
Crear un software para agendar consultas, revisiones y exámenes en una clínica odontológica, con dos doctores trabajando con los mismos horarios.

Tecnologías:
✅Backend: .NET (C#)
✅Frontend: WPF
✅Base de datos: PostgreSQL

Funcionalidades:
✅Agendar citas por fecha y hora
✅Ver la disponibilidad de cada doctor
✅Tener un calendario para la planificación
✅CRUD (Crear, Leer, Actualizar, Eliminar) para las citas
✅Ingreso de datos como nombre del paciente, tipo de consulta, doctor asignado
